== Description ==

Insert [Binlayer](http://binlayer.com/ref-194878.html "Binlayer") Layer Ad to your Blog. You can do great settings for different kind of pages etc. if and which ad should be inserted. 

Settings are self explanatory. 

If you like my work it would be great if you create a new [Binlayer](http://binlayer.com/ref-194878.html "Binlayer") account by using [this](http://binlayer.com/ref-194878.html "Binlayer") link. Costs you nothing, helps further development of this and other Plugins. 

If you also want to use Adsense and Wordpress i suggest the [Adsenseoptimizer Plugin](http://wordpress.org/extend/plugins/adsenseoptimizer/ "Adsenseoptimizer"). 

If you want to show your BinLayer Earnings in your Dashbord, you can use [this Plugin](http://wordpress.org/extend/plugins/binlayer-dashboard-widget/ "BinLayer Dashbord Widget").
